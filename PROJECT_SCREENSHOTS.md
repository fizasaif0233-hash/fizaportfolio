## How to Add Screenshots

1. Locate the screenshots in your temp directory (they were saved during the browser session)
2. Copy all PNG files to `public/images/` directory
3. Ensure the filenames match exactly as listed above
4. The portfolio will automatically display the images

## Alternative: Re-capture Screenshots

If you prefer to capture fresh screenshots:
1. Visit each project URL
2. Take full-page screenshots
3. Save them to `public/images/` with the exact filenames listed above

All project components are already configured to display these images when they're available in the `public/images/` directory.
